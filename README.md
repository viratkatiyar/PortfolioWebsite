# Virat Katiyar — Portfolio 🚀

Personal portfolio website built with React, TypeScript, GSAP, and Three.js.

---

## 🛠️ Instructions

GSAP Club plugins are used via trial versions in this repo.  
⚠️ Trial plugins cannot be used for production or hosting.

For official GSAP Club plugins:  
https://gsap.com/docs/v3/Installation/

---

## ⚙️ Tech Stack

React • TypeScript • GSAP • Three.js • WebGL • HTML • CSS • JavaScript

---

## 🎨 Assets

Some 3D assets are included for reference. The original custom 3D avatar used on the live site is **not** included and is not open source.
